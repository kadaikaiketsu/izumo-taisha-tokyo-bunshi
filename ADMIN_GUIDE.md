# 📰 新着情報 管理ツール 使い方ガイド

出雲大社東京分祠の新着情報を簡単に管理できるツールです。

---

## 🚀 アクセス方法

### 方法1: news.htmlから
1. `news.html`（新着情報ページ）を開く
2. 右上の「管理画面」ボタンをクリック

### 方法2: 直接アクセス
- ブラウザで `admin.html` を開く

---

## 📋 機能一覧

### ✅ できること

1. **新着情報の追加**
   - タイトル、日付、本文を入力して追加

2. **既存記事の編集**
   - 記事の修正・更新

3. **記事の削除**
   - 不要な記事を削除

4. **プレビュー機能**
   - 公開前に表示を確認

5. **統計情報**
   - 記事数、最終更新日を表示

---

## 📝 使い方

### 1️⃣ 新しい記事を追加

#### ステップ1: 管理画面を開く
```
admin.html を開く
```

#### ステップ2: フォームに入力
```
日付: 2025-12-14  （カレンダーから選択）
タイトル: 令和7年 神前結婚式のご案内
内容: 神前結婚式を執り行っております。
      詳しくはお電話にてお問い合わせください。
```

#### ステップ3: プレビュー（任意）
```
「プレビュー」ボタンをクリック
→ 表示を確認
→ OKなら「×」で閉じる
```

#### ステップ4: 追加
```
「記事を追加」ボタンをクリック
→ 「記事を追加しました！」と表示
→ 新着情報ページに自動反映
```

---

### 2️⃣ 記事を編集

#### ステップ1: 編集したい記事を探す
```
「既存の記事一覧」セクションまでスクロール
```

#### ステップ2: 編集ボタンをクリック
```
記事の右下「編集」ボタンをクリック
→ フォームに内容が読み込まれる
```

#### ステップ3: 内容を修正
```
タイトル、日付、内容を修正
```

#### ステップ4: 更新
```
「記事を更新」ボタンをクリック
→ 「記事を更新しました！」と表示
```

---

### 3️⃣ 記事を削除

#### ステップ1: 削除したい記事を探す
```
「既存の記事一覧」セクションまでスクロール
```

#### ステップ2: 削除ボタンをクリック
```
記事の右下「削除」ボタンをクリック
→ 確認ダイアログが表示
```

#### ステップ3: 確認
```
「OK」をクリックで削除
「キャンセル」をクリックで中止
```

---

## 💡 便利な機能

### 📅 日付の入力
- カレンダーから選択できます
- デフォルトで今日の日付が設定されます

### 📝 改行について
- 内容欄で改行すると、そのまま表示されます
- 箇条書きも可能です

### ⌨️ キーボードショートカット
- `Ctrl + S` または `Cmd + S`: 記事を保存
- `ESC`: プレビューを閉じる

### 🔄 自動保存
- データはブラウザに保存されます
- ページを閉じても消えません

## ⚠️ 注意事項

### データの保存について
- **ブラウザに保存**: データはブラウザのLocalStorageに保存されます
- **ブラウザごと**: ChromeとSafariなど、ブラウザごとに別々のデータです
- **同じブラウザで**: 同じブラウザであれば、どのページからでも同じデータが見えます

### バックアップについて
- 定期的にスクリーンショットを取っておくと安心です
- 重要な記事は別途テキストファイルに保存することをおすすめします

### ブラウザを変える場合
- データは引き継がれません
- 再度入力が必要になります

## 🔧 トラブルシューティング

### Q: 記事が表示されない
**A**: 以下を確認してください
1. ブラウザの更新（F5キー）
2. キャッシュのクリア
3. 別のブラウザで確認

### Q: データが消えた
**A**: 考えられる原因
1. ブラウザのキャッシュをクリアした
2. 別のブラウザで開いている
3. プライベートモード/シークレットモードで開いている

### Q: 追加ボタンを押しても反応しない
**A**: 確認事項
1. すべての項目（日付、タイトル、内容）が入力されているか
2. ブラウザのエラーコンソールを確認（F12キー）

- 入力がしやすい

## ✨ 活用のヒント

### 定期的な更新
```
おすすめ頻度:
- 月1回: 行事のお知らせ
- 随時: 臨時のお知らせ
- 年数回: 初穂料改定などの重要なお知らせ
```

### タイトルの付け方
```
良い例:
✅ 「令和7年 年越の大祓のご案内」
✅ 「◆御祈祷初穂料改定のお知らせ」
✅ 「臨時休業のお知らせ（12月20日）」

わかりにくい例:
❌ 「お知らせ」
❌ 「重要」
❌ 「ご案内」
```

### 本文の書き方
```
ポイント:
1. 簡潔に
2. 日時・場所を明記
3. 問い合わせ先を記載（必要に応じて）
4. 改行を活用して読みやすく
```
