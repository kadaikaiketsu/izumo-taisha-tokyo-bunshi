## 完成した機能

### 全10ページの構成

1. **index.html** - トップページ
   - 分祠の紹介
   - サービス案内（御祈願、神前結婚式、御札・御守）
   - 基本情報（住所、電話、受付時間、アクセス）

2. **news.html** - 新着情報
   - 令和7年の年越の大祓のご案内
   - 御祈祷初穂料改定のお知らせ（令和7年4月15日より）
   - 挙式初穂料改定のお知らせ（令和5年1月1日より）

3. **about.html** - 出雲大社について
   - 出雲大社（いずもおおやしろ）の説明
   - 出雲大社教（いずもおおやしろきょう）について
   - 出雲大社と出雲大社教の歴史

4. **history.html** - 由緒
   - 出雲大社東京分祠の歴史（明治11年～現在）
   - 神前結婚式の歴史
   - 国葬・神葬祭の奉仕歴

5. **omamori.html** - 御札・御守
   - 御神札（御玉串、御木札、関札、釜社札）
   - 縁結び（縁結守、縁むすびの糸）
   - 御守その他（壮気健全守、厄除守、開運守、しあわせの鈴、交通安全守、学業守、長寿守、安産守）
   - 出雲大社の神札についての注意事項

6. **prayer.html** - 御祈願
   - 御祈祷の時間・初穂料
   - 主な御祈願（家内安全、商売繁盛、縁結、交通安全、初宮参り、七五三、厄除、出雲屋敷地鎮祭、その他）
   - 郵送での御祈願について

7. **wedding.html** - 神前結婚式
   - 御挨拶
   - 挙式について
   - 結婚式次第（参進、修祓、祝詞奏上、神誡、玉串拝礼、指輪の交換、夫婦盃の儀、親族盃の儀、退下）
   - 挙式御初穂料（80,000円）
   - 御衣裳のご案内
   - 出雲大社奉斎の結婚式場御案内

8. **events.html** - 年間行事
   - 令和7年の年間行事カレンダー
   - 月次祭、甲子祭、大祓、春季大祭、秋季大祭など

9. **funeral.html** - 神葬祭
   - 神葬祭について（幽顕一如の教え）
   - 神葬祭の流れ（移霊祭、葬儀、帰家祭、十日祭）
   - みたままつり（五十日祭、年祭、五十年祭）
   - 拝礼の作法（玉串拝礼）
   - 出雲大社が勤仕した主な国葬・国民葬

10. **access.html** - 交通アクセス
    - 地下鉄でのアクセス（東京メトロ日比谷線、都営地下鉄大江戸線）
    - 基本情報（住所、電話、受付時間、御祈祷時間）
    - お車でお越しの方への案内
    - 周辺情報
